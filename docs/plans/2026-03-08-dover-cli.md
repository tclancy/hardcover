# Dover Library CLI — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build an interactive Python CLI that picks a random book from the user's Hardcover "Want to Read" list, searches Dover Public Library's Koha catalog, and records yes/no match decisions to build a confidence-scored matching system.

**Architecture:** A Typer CLI with two commands — `sync` (fetch and cache Hardcover lists locally) and `check` (pick a random book, run the two-phase Dover search, prompt for confirmation, store the decision). All data lives in a local SQLite DB. The confidence scorer is a weighted rule-based function seeded by the existing "In Dover" list as known-good training examples.

**Tech Stack:** Python 3.12+, uv, typer, httpx, beautifulsoup4, python-dotenv, difflib (stdlib), sqlite3 (stdlib)

---

## Background & Key Decisions

- **Hardcover API**: GraphQL at `https://api.hardcover.app/v1/graphql`. Auth via `Authorization: Bearer <token>`. Token stored in `.env` as `HARDCOVER_TOKEN`. The implementer should introspect the schema or check https://hardcover.app/account/api for the token page and any docs.
- **Dover Library (Koha OPAC)**: HTML search results — no JSON API. Parse with BeautifulSoup. Base URL: `https://librarycatalog.dover.nh.gov/cgi-bin/koha/opac-search.pl`
- **Two-phase search**:
  1. Title-only (stop words stripped). Count results.
     - 0 results → record "not found", done
     - 1 result → show it, ask confirm (y/n/skip)
     - 2–10 results → show all, ask user to pick number or skip
     - 11+ results → proceed to phase 2
  2. Title + author last name. Same thresholds.
     - Still 11+ → show top 5 by confidence score, ask user to pick or skip
- **Stop words to strip**: `the, a, an, of, in, and, or, to, for, with, at, by, from, on, as`
- **Confidence score features** (0.0–1.0):
  - Exact title match (normalized): weight 1.0
  - Fuzzy title similarity (difflib.SequenceMatcher): weight 0.8
  - Author last name match: weight 0.6
  - Publication year match: weight 0.2
  - Final score = weighted sum / sum of weights (i.e. weighted average)
- **"In Dover" list**: a Hardcover list the user manually curates. These known-good pairs (Hardcover book → confirmed Dover match) are the training set for calibrating confidence weights later.
- **Truly random**: `random.choice(books)` — no processed/unprocessed tracking. Intentional, to detect if search results change over time.

```python
import sqlite3
from pathlib import Path
from typing import Any

DB_PATH = Path(__file__).parent.parent / "data" / "hardcover.db"


def get_connection(path: Path = DB_PATH) -> sqlite3.Connection:
    conn = sqlite3.connect(path)
    conn.row_factory = sqlite3.Row
    return conn


def init_db(conn: sqlite3.Connection) -> None:
    conn.executescript("""
        CREATE TABLE IF NOT EXISTS books (
            hardcover_id INTEGER PRIMARY KEY,
            title TEXT NOT NULL,
            author TEXT NOT NULL,
            slug TEXT,
            synced_at TEXT DEFAULT (datetime('now'))
        );

        CREATE TABLE IF NOT EXISTS in_dover (
            hardcover_id INTEGER PRIMARY KEY,
            title TEXT NOT NULL,
            author TEXT NOT NULL,
            slug TEXT,
            synced_at TEXT DEFAULT (datetime('now'))
        );

        CREATE TABLE IF NOT EXISTS searches (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            hardcover_id INTEGER NOT NULL,
            query TEXT NOT NULL,
            phase INTEGER NOT NULL,  -- 1 or 2
            result_count INTEGER NOT NULL,
            searched_at TEXT DEFAULT (datetime('now')),
            FOREIGN KEY (hardcover_id) REFERENCES books(hardcover_id)
        );

        CREATE TABLE IF NOT EXISTS decisions (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            hardcover_id INTEGER NOT NULL,
            koha_title TEXT,
            koha_author TEXT,
            confirmed INTEGER NOT NULL,  -- 1=yes, 0=no/skip
            confidence REAL,
            search_query TEXT,
            decided_at TEXT DEFAULT (datetime('now')),
            FOREIGN KEY (hardcover_id) REFERENCES books(hardcover_id)
        );
    """)
    conn.commit()


def upsert_book(conn: sqlite3.Connection, *, hardcover_id: int, title: str, author: str, slug: str) -> None:
    conn.execute("""
        INSERT INTO books (hardcover_id, title, author, slug, synced_at)
        VALUES (?, ?, ?, ?, datetime('now'))
        ON CONFLICT(hardcover_id) DO UPDATE SET
            title=excluded.title,
            author=excluded.author,
            slug=excluded.slug,
            synced_at=excluded.synced_at
    """, (hardcover_id, title, author, slug))
    conn.commit()


def upsert_in_dover(conn: sqlite3.Connection, *, hardcover_id: int, title: str, author: str, slug: str) -> None:
    conn.execute("""
        INSERT INTO in_dover (hardcover_id, title, author, slug, synced_at)
        VALUES (?, ?, ?, ?, datetime('now'))
        ON CONFLICT(hardcover_id) DO UPDATE SET
            title=excluded.title,
            author=excluded.author,
            slug=excluded.slug,
            synced_at=excluded.synced_at
    """, (hardcover_id, title, author, slug))
    conn.commit()


def get_all_books(conn: sqlite3.Connection) -> list[dict[str, Any]]:
    return [dict(row) for row in conn.execute("SELECT * FROM books").fetchall()]


def get_in_dover_books(conn: sqlite3.Connection) -> list[dict[str, Any]]:
    return [dict(row) for row in conn.execute("SELECT * FROM in_dover").fetchall()]


def record_decision(conn: sqlite3.Connection, *, hardcover_id: int, koha_title: str | None,
                    koha_author: str | None, confirmed: bool, confidence: float | None,
                    search_query: str) -> None:
    conn.execute("""
        INSERT INTO decisions (hardcover_id, koha_title, koha_author, confirmed, confidence, search_query)
        VALUES (?, ?, ?, ?, ?, ?)
    """, (hardcover_id, koha_title, koha_author, int(confirmed), confidence, search_query))
    conn.commit()


def get_decisions_for_book(conn: sqlite3.Connection, *, hardcover_id: int) -> list[dict[str, Any]]:
    return [dict(row) for row in conn.execute(
        "SELECT * FROM decisions WHERE hardcover_id = ? ORDER BY decided_at DESC",
        (hardcover_id,)
    ).fetchall()]
```

```python
import difflib
import re

STOP_WORDS = {"the", "a", "an", "of", "in", "and", "or", "to", "for", "with", "at", "by", "from", "on", "as"}

WEIGHTS = {
    "title_exact": 1.0,
    "title_fuzzy": 0.8,
    "author_last": 0.6,
    "year": 0.2,
}
TOTAL_WEIGHT = sum(WEIGHTS.values())


def strip_stop_words(title: str) -> str:
    words = title.split()
    filtered = [w for w in words if w.lower() not in STOP_WORDS]
    return " ".join(filtered) if filtered else title


def normalize_title(title: str) -> str:
    title = title.strip().lower()
    title = strip_stop_words(title)
    title = re.sub(r"[^\w\s]", "", title)
    return title


def _extract_last_name(author: str) -> str:
    """Handle both 'First Last' and 'Last, First' formats."""
    author = author.strip()
    if "," in author:
        return author.split(",")[0].strip().lower()
    parts = author.split()
    return parts[-1].lower() if parts else author.lower()


def score_match(*, hc_title: str, hc_author: str,
                koha_title: str, koha_author: str,
                koha_year: int | None, hc_year: int | None = None) -> float:
    norm_hc = normalize_title(hc_title)
    norm_koha = normalize_title(koha_title)

    title_exact = 1.0 if norm_hc == norm_koha else 0.0
    title_fuzzy = difflib.SequenceMatcher(None, norm_hc, norm_koha).ratio()

    hc_last = _extract_last_name(hc_author)
    koha_last = _extract_last_name(koha_author)
    author_last = 1.0 if hc_last == koha_last else 0.0

    year_score = 0.0
    if hc_year and koha_year:
        year_score = 1.0 if hc_year == koha_year else 0.0

    weighted = (
        WEIGHTS["title_exact"] * title_exact
        + WEIGHTS["title_fuzzy"] * title_fuzzy
        + WEIGHTS["author_last"] * author_last
        + WEIGHTS["year"] * year_score
    )
    return weighted / TOTAL_WEIGHT
```

**Step 4: Implement `cli/dover.py`**

```python
import httpx
from bs4 import BeautifulSoup
from cli.scorer import strip_stop_words, STOP_WORDS

KOHA_BASE = "https://librarycatalog.dover.nh.gov/cgi-bin/koha/opac-search.pl"
KOHA_LIMITS = "&limit=itype%3ABK&limit=branch%3ADOVER"

# NOTE: These CSS selectors were identified from Koha OPAC HTML.
# If results look wrong, re-inspect the HTML with the curl command in Task 4 Step 1.
RESULT_CONTAINER = "li.col-sm-10.details"
TITLE_SELECTOR = "a.title"
AUTHOR_SELECTOR = "span.author"
YEAR_SELECTOR = "span.results_summary"


def strip_stop_words_from_query(title: str) -> str:
    return strip_stop_words(title)


def build_title_query(title: str) -> str:
    clean = strip_stop_words_from_query(title)
    from urllib.parse import quote_plus
    return f"{KOHA_BASE}?q={quote_plus(clean)}{KOHA_LIMITS}"


def build_author_query(title: str, author: str) -> str:
    from urllib.parse import quote_plus
    clean_title = strip_stop_words_from_query(title)
    # Extract last name for author search
    parts = author.strip().split()
    last = parts[-1] if parts else author
    query = f"{clean_title} {last}"
    return f"{KOHA_BASE}?q={quote_plus(query)}{KOHA_LIMITS}"


def parse_search_results(html: str) -> list[dict]:
    soup = BeautifulSoup(html, "html.parser")
    results = []
    for item in soup.select(RESULT_CONTAINER):
        title_el = item.select_one(TITLE_SELECTOR)
        author_el = item.select_one(AUTHOR_SELECTOR)
        title = title_el.get_text(strip=True) if title_el else ""
        author = author_el.get_text(strip=True) if author_el else ""
        # Extract year — Koha puts it in a results_summary span, may need tuning
        year = None
        for span in item.select(YEAR_SELECTOR):
            text = span.get_text()
            import re
            m = re.search(r"\b(19|20)\d{2}\b", text)
            if m:
                year = int(m.group())
                break
        results.append({"title": title, "author": author, "year": year})
    return results


def search(query_url: str, client: httpx.Client | None = None) -> list[dict]:
    """Fetch a Koha search URL and return parsed results."""
    if client:
        response = client.get(query_url)
    else:
        with httpx.Client(follow_redirects=True, timeout=15) as c:
            response = c.get(query_url)
    response.raise_for_status()
    return parse_search_results(response.text)
```

```python
def _graphql_post(query: str, variables: dict | None = None, token: str = "") -> dict:
    with httpx.Client(timeout=30) as client:
        resp = client.post(
            API_URL,
            json={"query": query, "variables": variables or {}},
            headers={"Authorization": f"Bearer {token}", "Content-Type": "application/json"},
        )
        resp.raise_for_status()
        return resp.json()


def extract_author_name(contributions: list[dict]) -> str:
    if not contributions:
        return "Unknown"
    return contributions[0]["author"]["name"]


def _parse_book(book: dict) -> dict:
    return {
        "hardcover_id": book["id"],
        "title": book["title"],
        "slug": book.get("slug", ""),
        "author": extract_author_name(book.get("contributions", [])),
    }


def fetch_want_to_read(token: str) -> list[dict]:
    data = _graphql_post(WANT_TO_READ_QUERY, token=token)
    user_books = data["data"]["me"]["user_books"]
    return [_parse_book(ub["book"]) for ub in user_books]


def fetch_in_dover_list(token: str, slug: str = IN_DOVER_LIST_SLUG) -> list[dict]:
    data = _graphql_post(IN_DOVER_QUERY, variables={"slug": slug}, token=token)
    lists = data["data"]["lists"]
    if not lists:
        return []
    list_books = lists[0]["list_books"]
    return [_parse_book(lb["book"]) for lb in list_books]
```

```python
RESULT_THRESHOLD_FEW = 10     # ≤10 results: show them all
RESULT_THRESHOLD_MANY = 10    # >10: trigger phase 2


def _display_results(results: list[dict], book: dict, limit: int | None = None) -> None:
    display = results[:limit] if limit else results
    for i, r in enumerate(display, 1):
        score = score_match(
            hc_title=book["title"], hc_author=book["author"],
            koha_title=r["title"], koha_author=r.get("author", ""),
            koha_year=r.get("year")
        )
        typer.echo(f"  [{i}] {r['title']} — {r.get('author', 'Unknown')} ({r.get('year', '?')})  score={score:.2f}")


def _ask_user(results: list[dict], book: dict, query: str, conn) -> None:
    if not results:
        typer.echo("  No results found.")
        from cli.db import record_decision
        record_decision(conn, hardcover_id=book["hardcover_id"],
                        koha_title=None, koha_author=None,
                        confirmed=False, confidence=None, search_query=query)
        return

    _display_results(results)
    typer.echo("\nEnter result number to confirm match, or 's' to skip:")
    choice = typer.prompt(">").strip().lower()

    from cli.db import record_decision
    if choice == "s" or choice == "":
        record_decision(conn, hardcover_id=book["hardcover_id"],
                        koha_title=None, koha_author=None,
                        confirmed=False, confidence=None, search_query=query)
        typer.echo("  Skipped.")
    else:
        try:
            idx = int(choice) - 1
            if 0 <= idx < len(results):
                picked = results[idx]
                score = score_match(
                    hc_title=book["title"], hc_author=book["author"],
                    koha_title=picked["title"], koha_author=picked.get("author", ""),
                    koha_year=picked.get("year")
                )
                record_decision(conn, hardcover_id=book["hardcover_id"],
                                koha_title=picked["title"], koha_author=picked.get("author"),
                                confirmed=True, confidence=score, search_query=query)
                typer.echo(f"  Confirmed: {picked['title']} (score={score:.2f})")
            else:
                typer.echo("  Invalid choice, skipping.")
        except ValueError:
            typer.echo("  Invalid choice, skipping.")


@app.command()
def check():
    """Pick a random book from Want to Read and search Dover Library."""
    conn = get_connection()
    init_db(conn)
    books = get_all_books(conn)

    if not books:
        typer.echo("No books cached. Run `dover sync` first.")
        raise typer.Exit(1)

    book = random.choice(books)
    typer.echo(f'\nChecking: "{book["title"]}" by {book["author"]}')
    typer.echo("─" * 50)

    # Phase 1: title only
    q1 = build_title_query(book["title"])
    typer.echo(f"Phase 1 search: {q1}")
    results1 = search(q1)
    typer.echo(f"  {len(results1)} result(s)")

    if len(results1) == 0:
        typer.echo("  No results found — recording as not found.")
        from cli.db import record_decision
        record_decision(conn, hardcover_id=book["hardcover_id"],
                        koha_title=None, koha_author=None,
                        confirmed=False, confidence=None, search_query=q1)
        return

    if len(results1) <= RESULT_THRESHOLD_FEW:
        _ask_user(results1, book, q1, conn)
        return

    # Phase 2: title + author last name
    typer.echo(f"  Too many results ({len(results1)}), trying phase 2...")
    q2 = build_author_query(book["title"], book["author"])
    typer.echo(f"Phase 2 search: {q2}")
    results2 = search(q2)
    typer.echo(f"  {len(results2)} result(s)")

    if len(results2) <= RESULT_THRESHOLD_FEW:
        _ask_user(results2, book, q2, conn)
    else:
        # Still too many — show top 5 by confidence score
        typer.echo(f"  Still many results — showing top 5 by confidence score:")
        scored = sorted(results2, key=lambda r: score_match(
            hc_title=book["title"], hc_author=book["author"],
            koha_title=r["title"], koha_author=r.get("author", ""),
            koha_year=r.get("year")
        ), reverse=True)
        _ask_user(scored[:5], book, q2, conn)
```
